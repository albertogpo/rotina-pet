# Rotina Pet v0.6.3

## Melhorias de interface mobile

- Pull to refresh no topo da página, recarregando os dados do aplicativo.
- Gradiente fixo sobre a área da barra de status para melhorar a legibilidade no iPhone.
- Espaçamento corrigido entre nome e espécie nos animais arquivados.
- Botões secundários da refeição mantidos lado a lado em telas pequenas, com rótulo conciso “Não servida”.
- Navegação do histórico reorganizada para manter as duas setas na mesma linha e centralizar “Voltar para hoje” abaixo da data quando necessário.

Esta versão não exige migração de banco de dados.
